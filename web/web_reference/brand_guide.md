**wavydance.ai**

**BRAND GUIDELINES - JELLY SEA EDITION**

Version 2.0 · June 2026

# 1\. Brand Essence

wavydance.ai is a unified gateway (API relay) for large language models, built for developers and IT teams. One endpoint, one key and one bill connect users to every major LLM provider.

## Brand idea

"The wave" is the current that carries every request to the right model and back. "The dance" is the effortless choreography of routing, failover and scaling behind one API. The Jelly Sea palette adds the feeling of a sunlit tropical lagoon: clear, translucent, effortless - infrastructure you can see straight through.

## Personality

- Fluent - everything flows; no friction, no lock-in.
- Transparent - clear water, clear pricing, clear metrics.
- Playful confidence - technical without being cold; a brand that moves.

# 2\. Slogan System

## Primary slogan

**One Wave. Every Model.**

Short, rhythmic, and ownable. "One wave" = one API call / one platform; "every model" = the entire LLM ecosystem.

## Supporting line (descriptor)

"The unified gateway to every LLM API." - use under the logo or in meta descriptions where the product must be explained instantly.

## Chinese counterpart (for future language switch)

一浪直达，万模归一 - "One wave reaches them all; every model in one place."

## Campaign / secondary lines

- Ride every model with one API.
- Change one line. Unlock every model.
- Built for the people who build.
- Clear as water. Fast as the current. (Jelly Sea campaign line)

# 3\. Logo

## Concept

The mark is a "W" drawn as a continuous flowing wave, ending in a bright relay node - the moment a request lands at the right model. A fainter echo wave underneath suggests motion and the dance partner.

## Color variants

- Dark mode: bright Jelly gradient (Jelly Azure → Jelly Turquoise → Ice Glass) on Deep Lagoon (assets/logo-mark.svg).
- Light mode: deeper Ink gradient (Lagoon Ink → Turquoise Ink) on Sea Foam, so it keeps contrast on white (assets/logo-mark-light.svg).

## Usage rules

- Primary lockup: mark + lowercase wordmark "wavydance" with ".ai" in the brand gradient.
- Clear space: keep at least the height of the relay node dot around the logo.
- Minimum size: 24 px height for the mark alone, 120 px width for the lockup.
- Never put the bright (dark-mode) gradient on a light background - switch to the Ink variant.
- Do not rotate, stretch, add shadows, or recolor the gradient.

## Accessibility

- Sea Foam Text on Deep Lagoon: ≈ 15:1 (AAA).
- Lagoon Deep on Sea Foam: ≈ 12:1 (AAA).
- Bright accents (Jelly Azure/Mint/Ice Glass) are for large text and UI only - never body copy on light backgrounds; use the Ink accents instead.
- Drift / Drift Ink meet AA for secondary text in their respective modes.

# 5\. Typography

## Typefaces

- Display / headlines: Space Grotesk (700, 500) - geometric with a technical personality.
- Body / UI: Inter (400-600) - neutral, highly legible at small sizes.
- Code / data: JetBrains Mono - code samples, API keys, metrics, kickers.

## Rules

- Headlines: tight letter-spacing (−1 to −2 px), sentence case with a period for slogans.
- Kickers / labels: JetBrains Mono, uppercase, +3 px letter-spacing, accent color of the active mode.
- Never use the gradient on long text - headline keywords only.

# 6\. Voice & Tone

- Speak developer-to-developer: concrete numbers ("<40ms overhead"), no marketing fluff.
- Water metaphors are seasoning, not the meal - at most one per screen or paragraph.
- Docs are dry and precise; landing pages and changelogs may dance a little.
- English first; every string must be i18n-ready for the planned language switcher.

# 7\. Applications

- Landing page: see index.html prototype - ships with both modes and a theme toggle (follows OS preference by default).
- Code blocks always stay dark (Deep Lagoon) in both modes, like sunlight hitting deep water.
- Product tiers carry wave names: Surf (free), Current (pay-as-you-go), Tsunami (enterprise).
- Status page language: "All systems flowing" instead of "operational".
- Social avatar: dark logo mark on Deep Lagoon rounded square (assets/logo-mark.svg).
